# PsFindFiles Module
## Description
Module for finding Office, media, graphics, and vault file types. Includes three public commands: Find-MsOfficeFile, Find-MediaFile, and Find-GraphicsFile, with legacy plural aliases.

## PsFindFiles Cmdlets
### [Find-MediaFile](Find-MediaFile.md)
Searches media (audio, video, picture) and vault files with optional recursion and CSV/JSON export.

### [Find-MsOfficeFile](Find-MsOfficeFile.md)
Searches Microsoft Office files (modern and optional legacy formats) with optional recursion.

### [Find-GraphicsFile](Find-GraphicsFile.md)
Searches 2D design files, 3D models, and point cloud assets with optional recursion and export.
