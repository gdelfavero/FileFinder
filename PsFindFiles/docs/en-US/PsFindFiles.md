# PsFindFiles Module
## Description
Module for finding Office, media, and vault file types. Includes two public commands: Find-MsOfficeFiles and Find-MediaFiles.

## PsFindFiles Cmdlets
### [Find-MediaFiles](Find-MediaFiles.md)
Searches media (audio, video, picture) and vault files with optional recursion and CSV/JSON export.

### [Find-MsOfficeFiles](Find-MsOfficeFiles.md)
Searches Microsoft Office files (modern and optional legacy formats) with optional recursion.
